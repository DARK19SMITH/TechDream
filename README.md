# TechDream

Plateforme web de vente et de conseils sur les gadgets informatiques.

🔗 [techdream.vercel.app](https://Techdream.vercel.app)

---

## Stack

- Next.js · TypeScript · CSS
- Déployé sur Vercel

## Lancer le projet

```bash
git clone https://github.com/DARK19SMITH/Techdream.git
cd Techdream
npm install
npm run dev
```

Ouvre [http://localhost:3000](http://localhost:3000) dans ton navigateur.
