## Project Summary
Site de vente et de conseil informatique "Tech Dream" proposant des solutions technologiques innovantes, des PC de haute performance et des conseils IT.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Database/Auth**: Supabase
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Runtime**: Bun

## Architecture
- `src/app`: Routes et pages de l'application (Boutique, Conseils, Contact, Admin).
- `src/components`: Composants UI réutilisables.
- `src/lib`: Configuration et utilitaires (Supabase client).
- `src/api`: Endpoints API pour les produits, articles et statistiques.

## User Preferences
- **Langue**: Français pour toute l'interface utilisateur.
- **Style**: Esthétique tech/futuriste avec des effets de particules et des patterns de circuits.
- **Composants**: Utilisation de composants fonctionnels React.
